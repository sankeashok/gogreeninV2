How to edit events:

  1. Find the event you want to rename below
  2. Copy the event name (e.g., "Event 1 - New Year Kickoff Ride")
  3. Open the corresponding year's JSON file (e.g., 2025/events.json)
  4. Replace the event name with your desired name
  5. Add URLs by adding: "url": "your-link-here" to each event

Example JSON format:
{"name": "Event 1 - Your Custom Name", "url": "https://facebook.com/post123", "photos": ["photo1.jpg"]}
